## 📖 Overview

**GroceryGo** is a web-based Grocery Delivery Management System developed using PHP and MySQL. The platform allows customers to browse grocery products, add items to their cart, place orders online, and track their purchases conveniently.

The system also provides an admin panel for managing products, categories, customers, and orders efficiently.

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/grocerygo.git
```

### 2. Move Project

Copy project folder to:

```text
xampp/htdocs/
```

### 3. Import Database

* Open phpMyAdmin
* Create a database named:

```sql
grocerygo_db
```

* Import the provided SQL file

### 4. Configure Database

Update database credentials in:

```php
config/database.php
```

### 5. Run Project

Start:

* Apache Server
* MySQL Server

Open:

```text
http://localhost/grocerygo
```
